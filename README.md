**專案功能概述：**
簡單的線上購物流程，整合了 LINE Pay 作為付款方式。

1.  **商品展示：**
    *   使用者可以瀏覽一個線上商品列表。
    *   商品包含名稱、價格等基本資訊。

2.  **購物車功能：**
    *   使用者可以將商品加入購物車。
    *   購物車會紀錄使用者選擇的商品及其數量。
    *   使用者可以清空購物車。
    *   在結帳時，會顯示購物車中所有商品的清單。

3.  **LINE Pay 整合：**
    *   使用者可以選擇使用 LINE Pay 完成付款。
    *   結帳時，應用程式會產生一個 LINE Pay 付款請求。
    *   使用者會被導向 LINE Pay 付款頁面。
    *   付款完成後，LINE Pay 會將使用者導回應用程式。
    *   應用程式會驗證付款結果。

**總結：**

這個專案提供了一個基本但完整的線上購物流程，整合了 LINE Pay 作為付款方式。
